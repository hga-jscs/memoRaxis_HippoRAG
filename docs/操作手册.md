# memoRaxis_HippoRAG 操作手册

## 1. 目标

本手册用于快速完成 HippoRAG 四任务实验，并记录统一 token 轨迹。

## 2. 环境

```bash
conda create -n memoraxis_hipporag python=3.11 -y
conda activate memoraxis_hipporag
pip install -r requirements.txt
```

准备配置文件：

```bash
cp config/config.example.yaml config/config.yaml
```

填写 `config/config.yaml` 中的：

- LLM `api_key`
- LLM `base_url`
- LLM `model`
- embedding `api_key`
- embedding `base_url`
- embedding `model`

## 3. 数据

将四个 parquet 文件放到：

```text
MemoryAgentBench/data/
```

执行：

```bash
python scripts/HippoRAG_MAB/data/convert_all_data.py
```

## 4. 最小测试

```bash
python run_all_tasks.py --acc_instance_idx 0 --conflict_instance_idx 0 --long_instance_idx 0 --ttl_instance_idx 0 --acc_limit 1 --conflict_limit 1 --long_limit 1 --ttl_limit 1 --run_id hippo_smoke_inst0 --skip_eval --force_rebuild
```

## 5. 标准运行

```bash
python run_all_tasks.py --acc_instance_idx 0 --conflict_instance_idx 0 --long_instance_idx 0 --ttl_instance_idx 0 --run_id hippo_all_inst0 --force_rebuild
```

## 6. 单任务运行

Accurate Retrieval：

```bash
python run_all_tasks.py --tasks acc --acc_instance_idx 0 --run_id acc_inst0 --force_rebuild
```

Conflict Resolution：

```bash
python run_all_tasks.py --tasks conflict --conflict_instance_idx 0 --run_id conflict_inst0 --force_rebuild
```

Long Range Understanding：

```bash
python run_all_tasks.py --tasks long --long_instance_idx 0 --run_id long_inst0 --force_rebuild
```

Test Time Learning：

```bash
python run_all_tasks.py --tasks ttl --ttl_instance_idx 0 --run_id ttl_inst0 --force_rebuild
```

## 7. 输出位置

索引：

```text
out/hipporag_indices/
```

推理结果：

```text
out/
```

评测结果：

```text
out/eval/
```

token 轨迹：

```text
out/token_traces/<run_id>.jsonl
```

## 8. 检查点

运行完成后，至少检查：

1. `out/token_traces/<run_id>.jsonl` 是否存在。
2. 四个结果 JSON 顶层是否有 `run_id` 和 `token_trace`。
3. Long Range 的评测结果是否写入 `out/eval/`。

## 9. 常见问题

- 找不到索引：先运行 ingest。
- 找不到数据：检查 `MemoryAgentBench/data/`。
- 配置报错：检查 `config/config.yaml`。
- 评测文件不存在：先确认 infer 已成功输出结果。
